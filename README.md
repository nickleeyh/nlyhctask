# Carousell Task

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

* Android Development Environments - Microsoft Visual Studio, IntelliJ IDEA, etc... - Android Studio preferred
* Android Device / Emulator

### Deployment

* Clone this repository into your local environment

* Open it with any Android Development Environment - Android Studio preferred

* Compile and build the project in the local environment

* Run the application into an emulator / android device
